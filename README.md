## ЁЯЪА Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ishagupta-https/team-terminal.git
```

### 2. Open the project

```bash
cd team-terminal
```

### 3. Run locally

Simply open:

```bash
index.html
```

or use a local server:

```bash
python -m http.server
```

Then visit:

```text
http://localhost:8000
```

## ЁЯОи Customization

### Add New Commands

Inside the `responses` object:

```javascript
responses["hello"] = [
  { t: "success", s: "Hello Developer!" }
];
```

### Add Hidden Commands

Inside the `hidden` object:

```javascript
hidden["secret"] = [
  { t: "meme", s: "You found the secret command!" }
];
```
